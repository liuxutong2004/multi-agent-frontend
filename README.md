# Enterprise Multi-Agent Business Analyzer 🚀

一个基于大语言模型（LLM）与多智能体（Multi-Agent）协同架构的自动化商业数据洞察与研报生成系统。通过复杂的长链推理机制，将传统企业内部耗时数天的数据提取、清洗、多维交叉归因分析与报告撰写流程，压缩至分钟级自动化闭环输出。

## 💡 架构设计与核心逻辑流
系统摒弃了单一模型直接输出的脆弱模式，采用深度协同架构，内置 4 个核心 Agent 相互制约与配合：

1. **Planner Agent (意图解析与规划)**：采用思维树（ToT）策略，将用户模糊的自然语言需求拆解为可执行的子任务链路，生成标准化流转程序。
2. **Data Engineer Agent (数据工程自纠错)**：结合企业表结构元数据自动编写查询语句，内置 Self-Correction 闭环纠错机制，遇报错自动重写。
3. **Analyst Agent (深度推理引擎)**：系统核心大脑，采用 ReAct 框架进行多步长链推理，对比历史数据并调用外部接口进行全网舆情交叉验证，深度推导问题背后的业务根因。
4. **Critic Agent (研报撰写与红队审查)**：将拓扑结构转化为长篇图文报告，同时作为红队进行严格的逻辑自洽性校验与幻觉（Hallucination）排查，确保绝对严谨后推送。
